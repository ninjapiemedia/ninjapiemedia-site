Ninja PIE Media website v3 upload instructions

Upload these files and folders to the root of the GitHub repository:

- index.html
- work.html
- collaborate.html
- links.html
- about.html
- contact.html
- privacy.html
- terms.html
- age.html
- release.html
- dmca.html
- compliance.html
- content-policy.html
- 404.html
- styles.css
- script.js
- assets/   (entire folder)

Do not upload the parent folder itself. Upload the contents into the repository root.

Cloudflare Pages will automatically redeploy after you commit the changes to GitHub.
